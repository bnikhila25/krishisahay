# Krishisahay

## Project Overview
Krishisahay is a project aimed at providing assistance in the field of agriculture. This platform is designed to connect farmers with resources, information, and services to enhance productivity and sustainability.

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Hosting:** AWS

## Features
- User Authentication
- Crop Management
- Resource Recommendations
- Real-time Weather Updates
- Community Forum

## Installation Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/bnikhila25/krishisahay.git
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the application:**
   ```bash
   npm start
   ```
